core_module_auto_key_app
========================

Auto Key module for the parser core project.

Quick start
===========

1. Add "core_module_auto_key_app" to your INSTALLED_APPS setting
----------------------------------------------------------------

.. code:: python

    INSTALLED_APPS = [
      ...
      'core_module_auto_key_app',
    ]

    PARSER_AUTO_KEY_KEYREF = True
